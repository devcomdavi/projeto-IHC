# AutoStore - Loja de Automóveis

Homepage responsiva para loja de automóveis desenvolvida em React com TailwindCSS.

## Componentes Implementados

### Elementos de Entrada:
1. **Combo boxes** - Seleção de país, estado e cidade (otimizado para filtros de localização)
2. **Toggle** - Alternância entre tema claro e escuro (posicionado no header para fácil acesso)
3. **Radio buttons** - Seleção de condição do veículo (novo/usado) nos filtros

### Elementos de Saída:
1. **Charts** - Timeline de preços usando Recharts para mostrar evolução de preços
2. **Alert UI** - Notificações de carrinho (produto adicionado/indisponível)
3. **Badges** - Contador de itens no carrinho (badge no ícone do carrinho)

### Navegação:
- **Menu hambúrguer** - Navegação entre marcas de automóveis (responsivo)

### Ajuda:
- **Tooltips** - Nos elementos de entrada para orientar o usuário

### Agrupamento:
- **Search bar** - Campo de pesquisa com opção de filtragem integrada

## Estrutura Responsiva

- **Header**: Logo, navegação principal, toggle de tema, carrinho e menu hambúrguer
- **Footer**: Informações da empresa, links rápidos e contato
- **Layout**: Totalmente responsivo (Web e Mobile)

## Como executar

```bash
npm install
npm start
```

## Tecnologias

- React 18
- TailwindCSS
- Recharts (gráficos)
- Lucide React (ícones)
